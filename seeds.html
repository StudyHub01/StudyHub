How to enter a Minecraft seed

  • Open Minecraft and start creating a new world

    From the main menu click Singleplayer then Create New World.

  • Find the seed field

    On Java Edition click More World Options. On Bedrock click Advanced. Both have a Seed field you can type into.

  • Paste the seed number

    Copy the seed from above and paste it. Include the minus sign if there is one -- it matters.

  • Check your game version

    Seeds work differently across versions. Make sure your Minecraft version is 1.21 or above for these seeds to generate correctly.

  • Create world and spawn in

    Click Create World. You will spawn at or near the locations described above.
